## 🐢해결법

git에서 user.name이나 user.email이 잘못 설정되어있을때 생기는 현상이다!🙀

git bash를 켜고 아래 명령어를 입력해보면 현재 어떤 정보가 들어가 있는지 확인할 수 있다🐧

``` git
git config --list
```

잘못된 정보를 수정하기 위해선 아래 명령어를 쳐주면 된다!👾

``` git
git config --global user.email "이메일"
git config --global user.name "이름"
```
